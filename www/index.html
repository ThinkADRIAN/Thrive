Universal WebView is an Xcode 6 project that allows you to wrap your website in a super simple iOS app. If you are a web developer who wants to release an iOS app, this should help you cut some corners when it comes to learning Swift. It is ideal for single page web app with on-screen/no navigation. For example, a game, interactive page, web based slideshow, etc.

Code is easily configurable and customizable. There is just one config file to setting up everything, simply edit the included UniversalWebView.plist file. You can also check a video manual. Create your own app in less than 15 minutes without any special knowledge! It's easier than you think.
